1 # The ASAP date/time library for beautiful C++ code
2 
3 ASAP is a small, header-only date-time library for C++11 and beyond. It is heavily inspired by my great time using [momentjs](momentjs.com) to parse and display dates. ASAP has a lot of syntatic sugar to make your code look pretty both on the inside and outside :)

7 ## Features
8 
9 - A date/time class to handle dates and time (duh)
10 - A duration class to handle unbounded periods (e.g, `asap::day(1)` specifies a day)
11 - A period class to handle bounded periods (e.g, `asap::period(asap::now(), asap::tomorrow())`)
12 - Lots of syntatic sugar (e.g, `auto then = asap::now() + asap::week(1)`)
13 - Literals (e.g, `auto then = asap::now() + 1_year`) for seconds, minutes, hours, days, weeks, months and years
14 - Broken down duration printing (e.g, `asap::days(10)` prints `"1 week, 3 days"`)
15 - Range-based iteration for periods:
16 ```
17  for (auto x: asap::now().until(asap::tomorrow()).every(1_hour)) {
18  std::cout << x << std::endl;
19  }
20 ```

29 ## Using
30 
31 You can either:
32 
33 - Drop the `include/asap` folder in your project and use it
34 - Make your compiler aware of the `include/asap` folder and include `"asap.h"` in your code
35 - If using CMake you can add the project subdirectory and link against it using `target_link_libraries`
36 - Use `find_package` if you installed ASAP using `make install` or any other method
37 - Download the latest release which is a single `asap.h` file bundled with all the classes.
